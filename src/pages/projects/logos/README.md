# Project Logos

Place your project logos in this folder with the following names:

- `workway-logo.png` - WorkWay platform logo
- `gaming-union-logo.png` - Israeli Gaming Union logo
- `hasbara-main-logo.png` - Main Hasbara website logo
- `hasbara-war-logo.png` - Iron Swords war hasbara logo
- `hasbara-me-logo.png` - Middle East hasbara logo
- `playoff-logo.png` - PlayOff platform logo

## Logo Requirements

- Format: PNG with transparent background
- Size: 60x60px (maximum, will be scaled to fit)
- High quality/resolution for crisp display

If a logo fails to load, a fallback cube icon will be displayed instead.
