# פקודות VIP ברובלוקס

רוצים לבאן.. או לפוצץ שחקן? הגעתם למקום הנכון.

:::info הערה חשובה
אנא ודאו שאתם בוחרים את התוסף הנכון מרשימה למטה. HD Admin ו-Kohl's Admin Infinite הם בעלי פקודות שונות.
:::

## איך להשתמש בפקודות HD Admin

1. לחצו על המקש `'` במקלדת שלכם
2. הקלידו פקודה בתיבת הטקסט ולחצו אנטר!

## פקודות כלים

### `;cmdbar`
**תיאור:** פותח את שורת הפקודות  
**דוגמה:** `;cmdbar`  
**כינויים:** commandBar  
**דרגה:** VIP

### `;refresh Player`
**תיאור:** מנקה את כל האפקטים והלולאות מעצמכם, או מהשחקן שצוין  
**דוגמה:** `;refresh builderman`  
**כינויים:** re, reset  
**דרגה:** VIP

### `;respawn Player`
**תיאור:** גורם לכם, או לשחקן שצוין, להיוולד מחדש  
**דוגמה:** `;respawn builderman`  
**כינויים:** res  
**דרגה:** VIP

### `;follow Player`
**תיאור:** מעביר אתכם לשרת בו נמצא השחקן שצוין (חייב להיות באותו משחק)  
**דוגמה:** `;follow builderman`  
**כינויים:** join, joinServer  
**דרגה:** VIP

## מידע נוסף

### תחביר פקודות
כל הפקודות עוקבות אחר הפורמט הסטנדרטי של HD Admin:
- פקודות מתחילות בנקודה-פסיק (`;`)
- שמות שחקנים יכולים להיות התאמות חלקיות
- השתמשו ב-"me" כדי למקד עליכם
- השתמשו ב-"all" כדי למקד על כל השחקנים (כשזמין)

### הערות חשובות

1. **נדרשת דרגת VIP**: כל הפקודות הרשומות למעלה דורשות דרגת VIP או גבוהה יותר
2. **HD Admin לעומת Kohl's Admin**: וודאו שאתם משתמשים במערכת האדמין הנכונה - הפקודות הללו מיועדות ל-HD Admin
3. **פקודות ביטול**: פקודות רבות כוללות גרסאות ביטול (מוצגות כשזמין)
4. **מיקוד שחקנים**: אתם יכולים למקד שחקנים לפי שם התצוגה או שם המשתמש שלהם

### קבלת גישת VIP

כדי לקבל גישת VIP בחוויית הרובלוקס שלנו:
1. הצטרפו לשרת הדיסקורד שלנו
2. בדקו את ערוץ #announcements עבור בקשות VIP
3. עקבו אחר תהליך הבקשה
4. חכו לאישור מהצוות

### הסבר קטגוריות פקודות

- **פקודות כלים**: פקודות חיוניות לפונקציות אדמין בסיסיות
- **פקודות מראה**: שינוי איך שחקנים נראים (בגדים, שינויי גוף)
- **פקודות אפקטים**: הוספת אפקטים ויזואליים לשחקנים
- **שליטה בשחקן**: שליטה בפעולות ומצבי שחקנים
- **מצלמה וממשק**: שינוי אלמנטי מצלמה וממשק
- **פקודות כיף**: פקודות בידור והומוריסטיות

### פתרון בעיות

**הפקודה לא עובדת?**
- וודאו שיש לכם את הדרגה הנכונה (VIP)
- בדקו את האיות והתחביר שלכם
- וודאו שהשחקן המטרה נמצא בשרת
- נסו להשתמש בשם המשתמש המלא במקום שם התצוגה

**האפקט לא מופיע?**
- אפקטים מסוימים עלולים להתנגש עם אחרים
- השתמשו בפקודת refresh כדי לנקות את כל האפקטים קודם
- בדקו אם לשחקן יש אפקטי חסינות פעילים

### תמיכה

אם אתם זקוקים לעזרה עם פקודות או נתקלים בבעיות:
